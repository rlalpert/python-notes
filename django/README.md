## New Project Setup

**New Directory**

`mkdir project`

**Setup Virtual Environment**

`python3 -m venv env`

**Activate VE**

`source env/bin/activate`

**Creating the Project**

`django-admin startproject mysite`

**Running the Development Server**

`python manage.py runserver`
